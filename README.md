# Linux scripts
These are scripts I use on my home machine for my own convenience.

# Manga

## Rename All
this file should be placed in your root download folder for manga-tui. It runs through
every directory and looks for available rename.sh or rename_weekly.sh files. It does not
look for rename_volume.sh, that you have to run manually

## Rename
This is the rename script for downloads from MangaDex (depending on how long that 
continues to be alive for). Should be placed in the same directory as the downloaded
files.

## Rename Weekly
This script is for all WeebCentral downloads that are weekly releases and not volume
based. Should be placed in the same directory as the downloaded files.

#- Rename Volume
Renames chapters downloaded from WeebCentral the same way as the MangaDex rename script
does. Should be placed in the same directory as the downloaded files.

# Utility

## Discordfix
This script is for the official Discord .deb release. When the .deb file is downloaded
simply run the script, and it will install the .deb file as well as download and run the
Vencord installer.
